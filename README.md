# DungeonFighter v2

Turn-based RPG in C# (.NET) with a data-driven action/combat system and an Avalonia UI.

## Quick start

- **Build**:
  - `cd Code`
  - `dotnet build`
- **Run**:
  - `cd Code`
  - `dotnet run`

## Where things are

- **Main code**: `Code/`
- **Game data**: `GameData/`
- **Architecture docs**: `Documentation/01-Core/ARCHITECTURE.md`
- **Settings system docs**: `Documentation/05-Systems/SETTINGS_SYSTEM_ARCHITECTURE.md`
- **Work tracking**: `TASKLIST.md`
- **Product overview**: `OVERVIEW.md`

## Tests

Unit tests live under `Code/Tests/Unit/`. The project includes in-game and CLI test runners; see existing docs in `Code/Tests/README.md`.
