The animations use the game's color markup system:
- {{W|text}} = White (main title text)
- {{R|text}} = Red (FIGHTER text)
- {{G|text}} = Green (borders and frames)
- {{Y|text}} = Yellow (prompt text)

Colors will display correctly in:
- Windows Terminal
- Windows Console with color support
- Most modern terminals
- Avalonia UI (when implemented)
